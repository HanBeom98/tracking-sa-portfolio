# Project Blueprint

## Overview

This project is a simple web page that displays a gallery of projects. It is built with HTML, CSS, and JavaScript, and it uses web components for the project cards. The page also includes a theme toggle for switching between light and dark modes, and a partnership inquiry form using Formspree.

## Features

*   **Project Gallery:** Displays a grid of project cards with images, titles, and descriptions.
*   **Web Components:** Uses a `project-card` web component to encapsulate the structure and styling of each project card.
*   **Theme Toggle:** Allows users to switch between a light and dark theme.
*   **Partnership Inquiry Form:** A form to receive partnership inquiries via Formspree.

## File Structure

*   `index.html`: The main HTML file, includes navigation and project gallery.
*   `inquiry.html`: The partnership inquiry form page.
*   `style.css`: The main stylesheet, including styles for both light and dark themes, and form styling.
*   `main.js`: The main JavaScript file, which includes the theme toggle logic and the `project-card` web component definition.
*   `blueprint.md`: This file.
