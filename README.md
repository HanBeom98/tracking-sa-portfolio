## 💎 Technical Mastery

### 1. Pragmatic DDD Architecture (1:1 Mirroring)
이론적인 아키텍처를 넘어, 번들러 없는 순수 JS 환경에 최적화된 **1:1 미러링 DDD** 구조를 채택했습니다.
- **Strict 4-Layer:** 모든 도메인은 `Domain`, `Application`, `Infra`, `UI` 레이어로 물리적/논리적으로 완벽히 격리됩니다.
- **Path Integrity:** 소스(`src`)와 빌드(`public`) 구조를 1:1로 일치시켜, 개발 환경의 무결성이 배포 환경으로 직접 전달되는 투명한 경로 체계를 유지합니다.
- **No-Magic Build:** 인위적인 코드 치환을 배제하고 파일 시스템의 직관성을 활용하여 런타임 안정성을 극대화했습니다.

### 2. Framework-less Premium UX
- **Web Components & Shadow DOM:** React/Vue 없이도 컴포넌트 기반의 독립적 개발과 완벽한 스타일 격리를 실현.
- **Traditional Board UI:** 정보 밀도와 가독성을 극대화한 **정통 리스트 형식**의 게시판을 구현, 최신 웹 표준 기술로 고전적 가치를 재해석.
- **Dynamic Theme Sync:** 사용자의 테마 환경을 실시간 감지하여 모든 게임 및 서비스의 시각 요소를 동적으로 동기화.

### 3. Continuous Integrity (Test-Driven)
- **88 Unit Tests:** 핵심 비즈니스 로직과 도메인 규칙을 88개의 단위 테스트로 촘촘히 검증.
- **E2E Smoke Testing:** GitHub Actions를 통해 실제 배포 사이트의 주요 기능을 자동 검증하여 '장애 없는 배포'를 실천.
