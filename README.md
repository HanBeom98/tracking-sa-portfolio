# 🚀 Tracking SA: High-Performance Data & AI Hub

> **"Shadow DOM 기반의 완벽한 캡슐화와 1:1 미러링 DDD 구조가 빚어낸 데이터 분석 및 AI 프리미엄 플랫폼"**

Tracking SA는 최신 웹 표준 기술(Web Components, OKLCH)과 엄격한 도메인 주도 설계(DDD)를 결합하여 탄생한 **넥슨 게임 전적 분석 및 지능형 라이프스타일 서비스 허브**입니다. 특히 서든어택 도메인은 프로젝트의 70% 이상을 차지하는 핵심 엔진으로 성장하였습니다.

---

## 💎 Technical Mastery

### 1. Pragmatic DDD Architecture (1:1 Mirroring)
이론적인 아키텍처를 넘어, 번들러 없는 순수 JS 환경에 최적화된 **1:1 미러링 DDD** 구조를 채택했습니다.
- **Strict 4-Layer:** 모든 도메인은 `Domain`, `Application`, `Infra`, `UI` 레이어로 물리적/논리적으로 완벽히 격리됩니다.
- **Sudden Attack Core:** 약 3,500 LOC 규모의 거대 도메인을 성공적으로 DDD로 관리하며, 팀 밸런싱 알고리즘 및 MMR 시스템을 구현했습니다.
- **No-Magic Build:** 인위적인 코드 치환을 배제하고 파일 시스템의 직관성을 활용하여 런타임 안정성을 극대화했습니다.

### 2. High-Performance Infrastructure
- **Vercel Edge & Serverless:** Vercel 환경에 최적화된 API 브릿지 설계를 통해 Cloudflare와 Vercel 양쪽에서 안정적인 AI 서비스를 제공합니다.
- **Cost-Effective Ops:** GitHub Actions의 무료 사용분을 보존하기 위해 수동 실행 및 스마트 트리거 시스템을 구축하여 지속 가능한 운영 환경을 확보했습니다.
- **ESM Native:** 모든 Functions와 API 레이어를 ESM(ES Modules) 규격으로 정규화하여 최신 Node.js 환경에서의 호환성을 극대화했습니다.
